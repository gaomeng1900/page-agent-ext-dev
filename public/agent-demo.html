🤖 Browser Agent Demo

📝 使用步骤：
  1. 点击扩展图标
  2. 点击 "打开 Agent 控制面板"
  3. 配置你的 OpenAI API Key
  4. 输入下面的示例任务
  5. 观察 Agent 自动执行

📋 示例任务

点击任意任务可以复制到剪贴板：

  • 打开 Google 并搜索 'browser automation'
  • Open GitHub, go to trending page, and show me the top repository
  • 打开知乎，搜索'人工智能'，点击第一篇文章
  • 在当前 tab 中向下滚动 3 次，然后返回顶部
  • Open https://example.com in a new tab and read the page title
  • 关闭所有标题包含 'test' 的标签页

🔧 可用工具

• open_tab - 打开新标签页
• active_tab - 切换到指定标签页
• close_tab - 关闭标签页
• reload_tab - 重新加载页面
• scroll_page - 滚动页面 (up/down/top/bottom)
• click - 点击元素（CSS 选择器）
• keydown - 键盘输入
• wait - 等待指定时间
• done - 标记任务完成

💡 高级示例

表单填写

在当前页面找到搜索框，输入 'AI agents'，然后点击搜索按钮

多步骤任务

1. 打开 https://github.com
2. 点击搜索框
3. 输入 'page-agent'
4. 点击搜索
5. 打开第一个结果

Tab 管理

查看所有打开的标签页，关闭所有不活动的标签页，只保留当前标签页

⚙️ 推荐配置

Model: gpt-4o (必须支持视觉输入！)
Base URL: https://api.openai.com/v1
API Key: sk-... (你的 OpenAI API Key)
✨ 提示：
  • 任务描述越具体，Agent 执行效果越好
  • Agent 可以"看到"页面截图，所以可以识别图像和布局
  • 复杂任务会消耗更多 tokens，注意成本
  • 第一次使用建议从简单任务开始
